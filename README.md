# AI CLI Assistant (Python)

A simple command-line AI assistant built in Python using the OpenAI API.  
This project demonstrates how to securely integrate an external AI API, send prompts, and display responses in a CLI environment.

## Features
- Command-line interface for interacting with an AI model
- Secure API key handling using environment variables
- Real-time prompt â†’ response interaction
- Basic error handling for missing keys or API issues

## Tech Stack
- Python
- OpenAI API
- python-dotenv
- CLI-based interaction

## How It Works
1. The user enters a prompt in the terminal
2. The application sends the prompt to the OpenAI API
3. The AI response is returned and displayed in the CLI
